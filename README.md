# مسبحة - سورة يس (Misbaha - Surah Yasin)

تطبيق بسيط لتلاوة سورة يس بأسلوب المسبحة، مصمم للتركيز والروحانية.

## المميزات

- تلاوة ميسرة بنظام الصفحات.
- تصميم داكن مريح للعين (Islamic Dark Mode).
- لا يوجد تتبع للوقت أو جمع للبيانات (خصوصية كاملة).
- إهداء خاص للروح الطاهرة.

## النشر على Vercel (Deployment)

يمكنك نشر هذا المشروع بسهولة على [Vercel](https://vercel.com):

### الخيار 1: باستخدام Vercel CLI

1. افتح مبنى الأوامر (Terminal) في مجلد المشروع.
2. قم بتثبيت Vercel CLI إذا لم يكن لديك: `npm i -g vercel`.
3. قم بتشغيل الأمر: `vercel`.
4. اتبع الخطوات في الشاشة.

### الخيار 2: الربط بـ GitHub

1. ارفع المشروع على مستودع (Repository) في GitHub.
2. اذهب إلى لوحة تحكم Vercel واربط المستودع.
3. سيتم النشر تلقائياً عند كل "Push".
